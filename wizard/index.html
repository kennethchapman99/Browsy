Step 5 of 10 · Variables

What changes each run?

Declare all variables the automation needs. Input = provided before the run. Captured = extracted from the page during the run (e.g. a generated ID in the URL). Derived = computed from input + captured.

Input variables — provided before the run (become the manifest)
Captured variables — extracted from the page during the run (not in manifest)
Derived variables — built from input + captured using {{varName}} templates
